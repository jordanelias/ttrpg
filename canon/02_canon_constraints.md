# Valoria Canon Constraints — Updated
## Extracted from: Valoria_Philosophical_Foundations.docx
## Version: 2026-04-02 (P-15 added from Amendment 01: three-layer being-persistence, Leap mechanics, Coherence 0)

Every mechanic, rule, and content element must satisfy all applicable constraints.
Violation = mechanical revision required. No exceptions.

| ID | Constraint | Foundations Ref | Mechanical Implication | Violation Test |
|----|-----------|----------------|----------------------|----------------|
| P-01 | **Inseparability**: all three dimensions (temporal, epistemic, actualized) co-move. Foregrounding is impossible. | §1.1 | Every thread operation MUST produce automatic co-movement effects across all three dimensions: temporal auto-effect (CD + History Resonance), epistemic auto-effect (investigation/testimony/Certainty modifiers), actual auto-effect (d6 consequence table). No GM discretion to skip. | Does the mechanic allow a thread op to resolve without all three dimensional auto-effects firing? If yes → FAIL. |
| P-02 | **Ein Sof = infinite positive being**: the unintelligible ground is fullness, not void or chaos. | §2.1 | Monstrous incursions are eruptions of *surfeit*, not disorder. Monster stat blocks must not imply evil, corruption, or negativity as ontological source. | Does any monster description frame its origin as chaos, void, or evil? If yes → FAIL. |
| P-03 | **Rendering = consciousness-performed**: rendering is what minds do, not an external mechanism. | §3.1–3.2 | GM is the rendering engine. Information asymmetry between sensitive and non-sensitive characters is the core mechanic, not an overlay. | Does the mechanic treat the rendered world as separate from consciousness? If yes → FAIL. |
| P-04 | **Monstrosity = ontological, not moral**: monstrous beings are rendering failures, not villains. | §4.1–4.2 | No alignment system. Monstrous entities are threats because they exceed rendering capacity, not because they are evil. Practitioners "manage rendering failure," not "fight evil." | Does any rule assign moral valence to monstrous entities? If yes → FAIL. |
| P-05 | **Three emergence modes are mechanically distinct**: Mode 1 (Ordinary), Mode 2 (Providence), Mode 3 (Threadcut). | §5.1–5.3 | Each mode must have distinct encounter procedures and persistence rules. Mode 1: no self-maintenance, deteriorates. Mode 2: temporary organization event, no entity stat block (D-18). Mode 3: continuous Thread work, indefinite persistence. | Are all three modes mechanically distinguishable? If any two are interchangeable → FAIL. Is Mode 2 treated as an entity rather than an event? If yes → FAIL. |
| P-06 | **Threadcut = radically is without becoming**: threadcut beings do not participate in organic temporal accumulation. | §5.3 | Past-Pull against threadcut being auto-produces Gap. Wounds cost sustained Thread work, not Ob penalty. Taint track does not apply. No aging, no organic change. | Does the mechanic treat threadcut beings identically to organic beings? If yes → FAIL. |
| P-07 | **Calamity = rendered-side mechanism**: over-drawing threads, not ground responsiveness. | §6.1 | TT system must model tension in the substrate from practitioner operations. The ground does not "respond" or "fight back." The fabric strains and tears. | Does any rule attribute agency, intention, or responsiveness to the Ein Sof or the ground? If yes → FAIL. |
| P-08 | **Epistemological barrier = inaccessibility, not suppression**: knowledge transmission is "religious poetry." | §10.1–10.2 | Inert Knowledge mechanic: non-sensitives can recite but not act with Thread-level precision. The barrier is metaphysical, not institutional (though Church reinforces it). Publishing all documents would not end the barrier. | Does any mechanic allow non-sensitives to gain Thread-level knowledge through study alone? If yes → FAIL. |
| P-09 | **Memory pulling = messy, costly, detectable**: not a clean eraser. | §13.1 | Memory pulling displaces the configuration, producing an orphaned configuration. If yes → FAIL. |
| P-10 | **Epistemic seduction = perceptual shift**: transformation dissolves categories, not corrupts the soul. | §16.1–16.3 | Taint track must not frame transformation as moral corruption. | P-11 | **Temporal Disjunction is universal**: all thread ops produce some TD, not just Past-Pulls. | §1.3 | Under Version C: every Thread operation produces an automatic temporal auto-effect (CD accumulation at minimum). | P-15 | **Three-layer being-persistence**: organic beings maintain themselves through (1) Ein Sof spooling, (2) unconscious self-rendering, (3) deliberate threadwork.

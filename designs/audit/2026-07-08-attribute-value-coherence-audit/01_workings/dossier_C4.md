# Dossier C4 — Tracks & Clocks (agonist pass)

**Headline.** The census correctly maps most of the surface noise in this cluster (IP name-split, WA/WR drift, VTM dead-but-gating, Turmoil/Strain self-disclosed alias, Disposition as the positive-control ED-912 propagation), but it missed the one file that actually claims and is actually *cited* as this cluster's single source of truth — `designs/provincial/clock_registry_v30.md` (self-declared `Status: CANONICAL`, "Single source of truth for all clocks, tracks, and counters," registered in `references/canonical_sources.yaml:419-423`, and named by file in `module_contracts.yaml` at four separate modules as the clock-owning authority) — which is itself demonstrably stale (a PP-498-superseded Torben Loyalty value still live on its own page) and absent from `CURRENT.md` entirely. Two of the census's own verdicts don't survive a direct read: **Renown is not an orphan** (it has a full definition at `derived_stats_v30.md §10.3`, already closed once by a dedicated 2026-05-10 orphan-terms sweep the census didn't check) and **Standing is not simply COHERENT** — it silently names two structurally unrelated quantities at two different scales (a 0–10 personal contest-kernel primitive and a 0–5 BG faction-political token track) that nothing in the corpus cross-references. The dossier's own name-collision probe pays off directly: **Bonds is a registered personal attribute (`attr.social.bonds`, descriptor_registry.yaml:44) *and* a `bucket: track` gate input in `module_contracts.yaml`'s fieldwork_knots module** (`g_bond5: "Bonds >= 5"`) with no cross-map between the two vocabularies — exactly the shape-divergence pattern the armature guardrail names, now confirmed at the stat layer. And Church Influence's renamed predecessor, "TC (Theocracy Counter)," is still carried as a second, separately-writable live clock in `module_contracts.yaml`'s territorial_piety module alongside CI itself, while the co-filed infill doc for that same module never received the CI rename at all — the same defect shape (struck/renamed quantity surviving as live register state) that `handoffs/HANDOFF_IN.md:177` already flagged once, uncited to any ED, for a different quantity (Coup Counter).

- **Confirmed, a real collision.** `references/descriptor_registry.yaml:44` registers Bonds as a base personal attribute: `{key: attr.social.bonds, name: Bonds, aliases: [Sincerity, Affability]}`, part of the 9-attribute (3 social) roster, scale 1–7 (registry header, line 30: `scale: "1-7"`).

Valoria · tag-anchored wiring · 9-tier

Wiring map — anchored on registry tags, not file paths

Every node hangs off a stable pointer/tag (module: from the 27-module contract · adapter: from cross-scale · key: from the type registry · ed: from the ledger · q: from the quantity registries) — so it survives file moves like the sim/→engine/ restructure. Structured on the 9-tier ladder: Wrapper → System → Subsystem → Module → Adapter → Mechanic → Submechanic → Routine → Subroutine. Two axes per unit: a build badge (this node's runtime wiring state) and a gd badge (the tagged unit's Godot-port state). Both, and every tag, are checked by tools/wiring_map_check.py --check against references/wiring_manifest.yaml — open ◧ Godot port view for the ranked port work-list.

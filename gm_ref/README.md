# gm_ref/

GM reference suite. Offsets tracking/accounting burden at the table. All content derived from mechanics — no editorial decisions embedded.

├── gm_reference_workplan.md   ← deliverable map and model assignments
├── dashboards/                ← D-01–D-10: GM screen tools, tracking forms, templates

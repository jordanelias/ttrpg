## NOTES FOR OPUS (D-11)

**On temporal beats:**
- Temporal shift ≠ aging or duration change. It is displacement of when this configuration belongs in the causal chain.
- TS tier differences must reflect ontical/ontological distinction (Foundations §17), not merely information quantity.
- [TARGET] placeholder must remain in all beats.

**On epistemic beats:**
- Epistemic shift ≠ forgetting. It is a change in epistemic accessibility to consciousness.
- The Ob modifier is already specified mechanically; the beat narrates *what that feels like*.
- [CONTEXT] placeholder for scene-specific detail.

**On d6 tables:**
- Each seed deliverable in one sentence.
- Seeds must be traceable to the operation type's logic — not arbitrary flavor.
- Seeds should be surprising but philosophically consistent.

**On Relational+ rows:**
- Knotted entity field: one sentence, TS-split where meaningful.
- The entity need not be present in the scene — the effect propagates through the knot.

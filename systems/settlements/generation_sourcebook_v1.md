## §1 The three parallel paradigm stacks

Each scale is generated by the same machine: a **vertical stack of weighted paradigms with noisy
throughlines** (seeded sampling; weights bias, noise chooses; coherence conditioned-in + threaded-in).
The three stacks nest: a **territory** seeds province-level scalars, then runs the **settlement** stack per
member (1–3); a **faction** is defined over the territories/settlements it holds.

### §1.1 SETTLEMENT stack (VSG — see `settlement_generator_v1.md` for the full spec)

P1 Location · P2 Starting faction · P3 Size · P4 Type · P5 Piety↔Restoration (Piety Track) ·
P6 Altonian influence · P7 Infrastructure · P8 Economy · P9 Caste/demographics · P10 Governance type ·
P11 Political allegiance · P12 Sociopolitical sympathies → P13 Actor roster (characters) · P14 Event deck
(events) · P15 Verify. Calibrated to reproduce Goldenfurt. Grounding now fully sourced (§2 below).

  Practitioner-Witness (heresy/Seam-Texts, TS≥30 comprehension-gate).
- **Altonia (P6/R1):** empire north of the mountain barrier; two passes (T3 Lowenskyst primary / T10 Spartfell
  secondary); Schoenland (T16) Altonian-trade island; naval route opens at IP≥75.
- **Caste (P9/R7):** Southern Einhir stigma originates in pre-Catastrophe southern (Lupicco) nexus-blame +
  northern-War-Coalition exclusion. Southern Einhir: higher baseline TS, economically suppressed; concentrated

- **Calamity (R6):** MS 0–100 (start ~72), radiates from Askeheim by node distance; hysteresis thresholds
  (fall 20/40/60, rise 28/48/68). d1 (T6/T13) = "the Borderlands," strongest Einhir folk practice + RM renewal.

## §3 Shared throughline constraints (what all three stacks must satisfy)

From the canonical framework (`throughlines_meta.md` + infill; `throughline_specifications_2026-04-17.md`):

- **М-2 Geography holds pressure** (T-18 Radiation Gradient, T-19 Southernmost) — a generated unit's geography
  must *distribute* the continuous clocks spatially (radiation creeps periphery→capital; frontier before seat).
- **М-4 Institutions stake substrate-postures** + **Reskinned-attractor** failure term — every faction (F1)
  and every institutional paradigm must be structurally distinct, not palette-swapped.
- **М-5 Scales connect through substrate** — each generated unit must specify how its state feeds the upward
  echo chain (settlement→territory→faction→peninsula), or it commits the **Scale-break** failure (violates Μ-δ).
- **М-6 Choice is forced** + **Q-robust dramatic-legibility triad** — a generated settlement must let a
  Valoria-literate designer answer in one sentence each: *whose position is at risk · what each named actor
  wants · what happens if no one acts next season*. (This is the acceptance bar for P13/P14 collision-wiring.)
- **§8.2-A subtractive doctrine (RATIFIED):** if two paradigms encode the same dynamic under different flavor
  → **MERGE/DISTILL**, never silent duplication; judge as-if-built; every cut survives an adversarial steelman.
- **Failure terms to avoid by construction:** Rest-state (a "solved" settlement must still move — Π anti-stall);
  Dominant-strategy / Flavor-only / Scale-break / Authored-emergence (Μ-β — no pre-scripted "emergent" events).

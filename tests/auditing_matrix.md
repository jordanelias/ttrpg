## NPC Rotation Schedule

For each scenario in S-01 through S-12, rotate named NPCs through the following roles:

| Role | Description |
|------|-------------|
| Primary actor | The NPC driving the test action |
| Opponent | The NPC opposing or being targeted |
| Witness/bystander | NPC present but not acting (tests co-movement blast radius, social consequences) |
| Absent stakeholder | NPC affected by cascade but not present (tests Domain Echo, faction ripple) |

**Minimum coverage:** Each named NPC must appear as Primary Actor in ≥3 scenarios, as Opponent in ≥2 scenarios, and as Witness in ≥1 scenario.

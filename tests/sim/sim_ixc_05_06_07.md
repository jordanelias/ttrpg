## IXC-06: Attributes → Threadwork

### Mode A — Attribute Group Analysis

Thread operations span three attribute groups: Spirit (Metaphysical) + Attunement (Social) + Focus (Mental). No single-group character can optimise all three.

### Mode B — Build Comparison (TS=50, Ob1 TN7)

| Build | Pool | P(success) | P(overwhelming) | Op rounds |
|-------|------|-----------|-----------------|-----------|
| Optimal (Spi=5,Att=4,Foc=4) | 15D | 91.2% | 74.0% | 3 |
| Budget (Spi=3,Att=3,Foc=3) | 12D | 87.8% | 64.7% | 2 |
| Minimum (Spi=1,Att=1,TS=30,Ob2) | 5D | 47.7% | ~8% | 0 |

Budget build saves 4 attribute points at cost of −5pp success, −15pp overwhelming, −1 op round. Efficient trade.

### Mode D — Edge Cases

**IXC-06-F1:** Thread ops require three different attribute groups. Pure-specialist build impossible by design.

**IXC-06-F2 [P2]:** Attunement double-valued: Leap pool contributor AND combat initiative advantage. Practitioner-combatants with high Attunement gain both. Concentrates power in single Social attribute.

**IXC-06-F3:** Focus=7 sequential failure: wastes 5 op rounds. High-Focus builds pay higher opportunity cost per failure. Mechanically sound — Focus is a concentration bet.

**IXC-06-F4:** Minimum practitioner (Spi=1,Att=1,TS=30): 47.7% Leap success. Marginal but legal.

**IXC-06-G1 [GAP]:** Maximum operations per contact window not stated. At Focus=7 (6 op rounds): can practitioner perform multiple ops per round?

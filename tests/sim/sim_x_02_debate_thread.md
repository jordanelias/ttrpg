# SIM-X-02: Debate + Thread (W-41 Conviction Anchor mid-Debate)
## Mode: C (Full Scenario) + B (Interaction Chain)
## Mechanics: §9 Debate (redesign v1) × W-41 (Conviction Anchor)
## Question: Does Thread use mid-debate produce degenerate outcomes or meaningful tactical choice?

---

## Setup

**Debate context:** Church tribunal. Inquisitor (Aldric) interrogating a practitioner-scholar (Vessa).
Asymmetric format: Inquisitor has procedural authority. Accused must satisfy inquisitor.
Audience: Church tribunal (values orthodoxy — Character/Ethos style favoured).

**Character: Vessa (practitioner-scholar)**
- Presence 4, Cognition 4, Spirit 4, Bonds 3, TS 70, Attunement 3
- Leading Belief: "The threads are not heresy — they are the world's constitution"
- Composure = Presence + Bonds = 4+3 = 7
- Debate pool base: Cognition + relevant History = 4+3 = 7D (TN7)
- Thread pool (W-41): Spirit + History + TPS = 4+2+7 = 13D, TN7
  - TPS = TS÷10 = 7

**Character: Aldric (Inquisitor)**
- Presence 5, Cognition 5, History (Canon Law) 4
- Debate pool: 5+4 = 9D (TN7)
- Composure = 9 (Presence + Bonds 4)
- Style: Character/Ethos (institutional authority; +2D on Character arguments before Church audience)

**Debate structure (from v1 redesign):**
Exchange format: each round, both declare style, roll, compare successes.
Win = deal Composure strain to opponent (1 per net success, minimum 1).
Rattled threshold at Composure ≤ 2 (−2D all rolls). Unmask = concede.

Audience modifier (Church tribunal, Character style): +2D to Character arguments for Aldric.
Effective Aldric pool (Character): 9+2 = 11D.

## State: Exchange 2

Vessa — Composure 6/7, Coherence 10/10
Aldric — Composure 9/9

**Decision point:** Vessa has taken 1 strain. She considers W-41 (Conviction Anchor).

**W-41 mechanic:**
- Requires the Leap (full-round, Priority 5; costs the practitioner their exchange action)
- Effect: Vessa's leading Belief gains +1D to resist rhetorical challenge for remainder of Debate
- This makes her leading Belief harder for Aldric to directly challenge (per debate v1 design: if Belief challenged directly, defender gets +Conviction dice)
- Coherence cost: Object/Personal scale = 0
- RS: unchanged on Success (Weaving)
- Thread pool: 13D TN7. P(Success/Overwhelming) ≈ ~99%

**Cost:** Vessa forfeits Exchange 2 entirely. Aldric gets a free exchange.

**Calculation — free exchange (Aldric unopposed):**
Unopposed Debate: 11D vs 0D = 11D net successes expected = ~3.6.
Composure strain to Vessa: 3–4 (expected 3.6).

**Decision problem:** Use W-41 now (take ~3.6 free strain) or wait (keep arguing, take ~1.3 strain/exchange)?

W-41 payoff: +1D to Belief resistance. Value = 1D×0.33 = ~0.33 extra net success per challenged exchange.
W-41 cost: ~3.6 strain now (one free exchange for Aldric).

Break-even analysis: 3.6 strain ÷ 0.33 per exchange = needs ~11 exchanges to recover cost.
**Finding F-08 [P2]:** W-41 is not worth using unless the Debate is expected to run 11+ exchanges AND Aldric directly challenges the Belief repeatedly. In a standard 5–7 exchange Debate, W-41 is always a net loss.

## State: Exchange 4 (Critical decision point)

Vessa — Composure 4/7 (approaching Rattled threshold at 2)
Aldric — Composure 9/9

**Vessa's options:**
1. Continue arguing: ~55–62% chance of losing each exchange, 1 strain/exchange → Rattled in ~2 more exchanges
2. Use W-41 now: takes ~3.6 free strain, immediately drops to Composure 0.4 → Rattled immediately. Not viable.
3. Unmask: concede the exchange, gain narrative leverage but lose Debate mechanically.

**Finding F-09 [P1 flag — design issue]:** In an asymmetric debate (practitioner scholar vs institutional authority with audience advantage), Vessa has no viable path to victory with 7D vs 11D effective. Expected drain rate: ~1 strain/exchange. From Composure 7, she reaches Rattled in ~5 exchanges and is effectively eliminated before she can mount a coherent response. W-41 does not address the structural problem (pool disparity + audience modifier). The Thread mechanic is decorative in this context.

**Root cause:** W-41 addresses the wrong problem. The practitioner's disadvantage is pool size and audience modifier, not Belief resistance. A +1D on Belief resistance only matters if Aldric challenges the Belief directly — which he doesn't need to do to win; he's winning on raw pool differential.

## Summary

| Finding | Category | Severity |
|---------|----------|----------|
| F-08 | Balance | P2 | W-41 break-even requires 11+ exchanges — exceeds typical Debate length |
| F-09 | Design | P1 | W-41 doesn't address structural disadvantage in asymmetric debates; practitioner has no Thread-based path to compensating pool disparity |
| F-10 | Interaction | P2 | Thread combat ops (W-20 series) and Debate Thread ops (W-40 series) have different cost structures; W-40 operations cost a full exchange action (high) vs W-20 costs a Leap round in combat (comparable); debate cost is proportionally higher |

**P1 Item:** W-41 is mechanically irrelevant in asymmetric debates. In symmetric debates (equal pools), +1D Belief resistance is marginal but not useless. The entire W-40 series may need a design pass to produce meaningful tactical choice.

[EDITORIAL: The P1 finding about W-41 ineffectiveness in asymmetric debates may require a design decision — either W-41 needs a broader effect (e.g., also grants +1D on any exchange involving the Belief's domain) or the Thread/Debate integration needs a different mechanic entirely. Requires user approval before any fix.]

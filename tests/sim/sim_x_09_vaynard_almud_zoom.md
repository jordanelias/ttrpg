## Exchange 1: Vaynard Consequence vs Almud Consequence

Both using Consequence style — no style mismatch for either. No resonant bonus (both are Consequence; each gets +1D when using their resonant style against their OWN audience — here they're debating each other, not an audience).

Wait — Resonant Style earns +1D when addressing the NPC *using their resonant style*. Vaynard's resonant style IS Consequence. Almud's resonant style IS Consequence. So:
- Vaynard using Consequence style arguing TO Almud (Consequence resonant): +1D.
- Almud using Consequence style arguing TO Vaynard (Consequence resonant): +1D.

Both get +1D. Symmetric bonus cancels.

Vaynard pool: 11+1=12D. Almud pool: 10+1=11D. Expected nets: 12×0.33=4.0 vs 11×0.33=3.6. Vaynard slight edge.

P(Vaynard wins Exchange 1): ~55%.
Most likely: Vaynard wins by 1 net success. Almud strain: +1. Composure 11→10.

### Zoom In: Discovery Event (mid-Exchange 2)

**Mechanical interrupt:** Discovery Event fires as an out-of-sequence event during Exchange 2 resolution. Per §11.3: personal → Thread transition requires the Leap; a Discovery Event is involuntary TS advancement, not a Leap. No Coherence cost. No RS cost. The transition is perceptual, not operational.

**Effect on Exchange 2:** Vaynard's argument about the Southernmost is suddenly informed by actual Thread perception. The GM may rule: +1D on Exchange 2 pool (the argument lands with unusual precision because Vaynard is now genuinely perceiving what he is describing). OR: no mechanical effect — the Discovery is internal, not rhetorically visible. GM call. More conservative reading: no bonus die, but the Debate's outcome now carries additional narrative weight.

Taking the conservative ruling (no bonus die). Exchange 2 resolves on existing pools.

Vaynard wins Exchange 2 (~55%). Almud strain: 2.
